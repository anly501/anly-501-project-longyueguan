What is API?

APIs is application programming interface and they are all around our life. It describes connectivity interfaces to an application. That means when we are use an app, they will connect the Internet and send data to server. And for our both Python section and R section, we use Twitter API to collect data from tweets with the specific topic. Since we are doing sentiment analysis of climate change, the topic that we choose when collect data with Twitter API is Climatchange.

The Twitter API enables programmatic access to Twitter in unique and advanced ways. And the Twitter Developer Platform allows us to harness the power of Twitter's open, global, real-time, and historical platform within our applications. And we use Twitter API to collect the most recent 100 tweets. After we get the API key, API key secret, Bearer Token, Access Token, and Access Token Secret, we can finish the set of collecting data.

References: “Twitter Developer Apps Overview | Docs | Twitter Developer Platform.” Twitter, Twitter, https://developer.twitter.com/en/docs/apps/overview.
